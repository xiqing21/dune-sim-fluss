**Slide 2：什么是 Spread 套利？（Basis Trade）****要点**：

- 跨交易所“现货 vs 永续合约”对冲套利
- 核心：**Delta Neutral（方向完全中性）**
- 只赚“基差收敛 + 资金费率”
- 不赌价格涨跌，只赚**相对价差**

**Slide 3：底层原理 1 —— 为什么 RAVE 这类币最适合？****要点**：

**Slide 7：资金费率（Funding Rate）的作用****要点**：

- 合约溢价（正基差）→ Funding **正** → 多头付空头（空头收钱）
- 现货溢价（负基差）→ Funding **负** → 空头短期付钱
- 实际操作中 Spread 利润通常远大于资金费率成本

**Slide 12：新增 —— 重要澄清：不是一定会现货涨到 1.8、合约到 1.6****核心问题解答**：

- **不是一定会**出现 1.8 / 1.6 这个具体价格
- 这只是**典型成功案例**的假设场景
- **底层逻辑**：
  - 在低流通、高控盘币中，庄家拉现货成本极低 → 现货供给极薄 → **现货溢价容易扩大**（基差 F - S 变负）
  - 永续合约 Mark Price 跟随现货 + 空头爆仓被动买盘 → 现货涨得更快，形成溢价
  - Funding Rate 机制也会推动基差向 0 收敛（正基差时空头赚钱，鼓励持仓）

**其他可能情况**：

1. **最佳情况**（如案例）：现货溢价扩大 → 基差收窄 → 盈利
2. **中性情况**：基差小幅震荡 → Funding 净收益覆盖成本
3. **亏损情况**：
   - 庄家弃盘或砸盘 → 现货溢价不扩大甚至反向（合约溢价持续）→ 基差扩大
   - Funding 长期为负且累计成本过高
   - 庄家短时极端砸盘 → ADL 强制平仓

**备注**：策略并非无风险，关键在于**小仓位 + 严格止损阈值 + 实时监控**。
